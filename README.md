# E-Commerce Product List

A modern, responsive e-commerce product listing application built with React and Material-UI. Features real-time search, category filtering, and persistent shopping cart functionality.

### Core Functionality
- **Product Display**: Responsive grid layout showcasing products with images, titles, prices, ratings, and descriptions
- **Category Filtering**: Filter products by category with visual chip selection
- **Search Capability**: Real-time search across product titles and descriptions with debounced input
- **Shopping Cart**: Add/remove products with persistent cart state and live counter badge
- **Responsive Design**: Fully responsive layout optimized for mobile, tablet, and desktop devices
